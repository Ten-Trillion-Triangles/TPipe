## Core Classes

### PcpContext
Defines the "Tool Belt" available to an agent.
- `transport`: Default transport (Stdio, Http, Python, Tpipe).
- `stdioOptions`, `tpipeOptions`, `httpOptions`, `pythonOptions`: Lists of allowed tools.
- `allowedDirectoryPaths`, `forbiddenFiles`: Filesystem restrictions.

### PcPRequest
A tool invocation generated by an LLM.
- `argumentsOrFunctionParams`: List of strings for arguments or script content.
- `stdioContextOptions`, `httpContextOptions`, etc.: Overrides for the specific call.

### PcpExecutionDispatcher
Executes requests and aggregates results.
- `executeRequest(request, context)`: Dispatches a single tool call.
- `executeRequests(requests, context)`: Runs multiple calls in parallel.

## Context Options

### StdioContextOptions
- `command`: Binary path or script name.
- `args`: Default parameters.
- `executionMode`: `ONE_SHOT`, `INTERACTIVE`, `CONNECT`, `BUFFER_REPLAY`.
- `permissions`: `Read`, `Write`, `Delete`, `Execute`.

### HttpContextOptions
- `baseUrl`, `endpoint`, `method`: URL and verb (GET, POST, etc.).
- `allowedHosts`:SSRRP protection allowlist.
- `authType`: `BASIC`, `BEARER`, `APIKEY`.

### PythonContext
- `availablePackages`: Whitelist of importable modules.
- `pythonPath`: Path to the local interpreter.

## Security Managers

### HttpSecurityManager
- `validateHttpRequest(options)`: Main entry point for HTTP security.
- **SSRF Protection**: Resolves hostnames once to prevent DNS rebinding. Blocks private IP ranges (127.x.x.x, 10.x.x.x, etc.) by default.

### PythonSecurityManager
- `validatePythonRequest(script, context)`: Main entry point for Python security.
- **AST Validation**: Uses `ast_validator.py` to parse script structure and block dangerous code (imports like `os`, `sys`, or functions like `eval`, `exec`) before execution.

## Standalone Hosting

### `PcpStdioHost`
- `runOnce()`: Processes one `PcPRequest` line and exits.
- `runLoop()`: Listens for requests on `stdin` until "exit".

### HTTP Hosting
Accessible via `POST /pcp` (Ktor).

## Result Classes

### PcpRequestResult
Outcome of a single tool call.
- `success`: Boolean status.
- `output`: String response or error details.
- `executionTimeMs`: Performance metric.

### PcpExecutionResult
Aggregate results for multiple tool calls.
- `results`: List of `PcpRequestResult`.
